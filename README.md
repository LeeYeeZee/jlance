## 概述

JLance 是一个 Java 实现的 [Lance](https://lancedb.github.io/lance/) 数据格式读取器，专门用于读取 **Lance v2.x** 列式数据文件并将其转换为 Apache Arrow `VectorSchemaRoot` 对象。该项目旨在与 Python Lance 实现保持跨语言兼容性。

目前项目为**只读实现**，尚不支持写入功能。

## Overview

JLance is a Java re-implementation of the [Lance](https://lancedb.github.io/lance/) data format reader. It targets reading **Lance v2.x** columnar data files and converting them into Apache Arrow `VectorSchemaRoot` objects. The project is designed for cross-language compatibility with the Python Lance implementation.

Currently, this is a **reader-only** implementation — there is no writer support yet.

- **Lance v2.0 / v2.1 文件格式** — 支持读取 footer、列元数据、全局缓冲区以及数据页  
  **Lance v2.0 / v2.1 File Format** — Reads footer, column metadata, global buffers, and data pages

- **V2.1 高级特性** — 支持 MiniBlockLayout、FullZipLayout、PackedStruct、字典编码、RLE 等  
  **V2.1 Advanced Features** — Supports MiniBlockLayout, FullZipLayout, PackedStruct, dictionary encoding, RLE, etc.

- **丰富的 Arrow 类型** — 支持 struct、list、nullable、binary、decimal、temporal、fixed-size list、large types 等  
  **Rich Arrow Types** — Supports struct, list, nullable, binary, decimal, temporal, fixed-size list, large types, etc.

- **多层嵌套** — 支持 `list<list<int>>`、`list<struct<...>>` 等复杂嵌套结构  
  **Deep Nesting** — Supports `list<list<int>>`, `list<struct<...>>`, and other complex nested structures

- **跨语言兼容测试** — 使用 Python Lance 生成的金标准文件进行 55+ 里程碑测试  
  **Cross-Language Compatibility** — 55+ milestone tests using golden files generated by Python Lance

### 读取 Lance 文件 / Read a Lance File

```java
import com.github.jlance.format.LanceFileReader;
import org.apache.arrow.memory.RootAllocator;
import org.apache.arrow.vector.VectorSchemaRoot;

public class Example {
    public static void main(String[] args) throws Exception {
        try (RootAllocator allocator = new RootAllocator();
             LanceFileReader reader = new LanceFileReader("data.lance")) {

            // 读取元数据 / Read metadata
            var metadata = reader.readMetadata();
            System.out.println("Rows: " + metadata.getNumRows());
            System.out.println("Schema: " + metadata.getSchema());

            // 读取所有数据 / Read all data
            try (VectorSchemaRoot root = reader.readBatch(allocator)) {
                System.out.println("Read " + root.getRowCount() + " rows");
                // 处理 Arrow 向量... / Process Arrow vectors...
            }
        }
    }
}
```

> **注意 / Note:** 在 Java 17+ 上运行测试需要添加 JVM 参数：  
> `--add-opens=java.base/java.nio=org.apache.arrow.memory.core,ALL-UNNAMED`

## Lance v2 文件布局 / Lance v2 File Layout

Lance 文件格式是极简的小端字节序结构：

- **数据页** → 任意缓冲区 / Data Pages → arbitrary buffers
- **列元数据** → Protobuf 消息 / Column Metadatas → protobuf messages
- **列元数据偏移表 (CMO)** → `(position, size)` 对 / Column Metadata Offset Table → `(position, size)` pairs
- **全局缓冲区偏移表 (GBO)** → `(position, size)` 对 / Global Buffer Offset Table → `(position, size)` pairs
- **Footer (40 字节, 小端)** / Footer (40 bytes, little-endian):
  - offset to column metadata
  - offset to CMO table
  - offset to GBO table
  - num global buffers
  - num columns
  - major/minor version
  - magic `"LANC"`

详见 `lance-core/src/main/proto/file2.proto` 中的 ASCII 图示。  
See the ASCII diagrams in `lance-core/src/main/proto/file2.proto`.
